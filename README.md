# L-aventure-de-fufus-le-lutin
Histoire exercice

-Il vit dans une cabane dans la foret de blackwood avec son chien Flipo.
Apres avoir terminé son labeur au champs, il rentra chez lui quand il fit une drôle de rencontre...
